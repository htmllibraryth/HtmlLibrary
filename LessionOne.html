บทเรียนที่ 1 เอชทีเอ็มแอลเบื้องต้น

  1. หน้าแรก
  2. บทเรียนที่ 1 เอชทีเอ็มแอลเบื้องต้น
  3. การสร้างและบันทึกเอกสาร
  4. การแสดงไฟล์เอกสารและการเปิดดู CODE ภาษา HTML

จุดประสงค์การเรียนรู้

1. สามารถอธิบายโครงสร้างและคำสั่งหลักของภาษา HTML ได้
2. อธิบายและใช้คำสั่งเบื้องต้นในการเขียนโปรแกรมภาษา HTML ได้
3. สามารถสร้างเว็บเพจด้วยภาษา HTML
4. สามารถแสดงผลลัพธ์บนเว็บบราวเซอร์ได้อย่างถูกต้อง
5. สามารถใช้โปรแกรม Notepad ในการพิมพ์ไฟล์เอกสาร HTML ได้

ภาษา HTML

     HTML หรือ HyperText Markup Language เป็นภาษาคอมพิวเตอร์รูปแบบหนึ่ง ที่มีลักษณะเป็นภาษาในเชิงการบรรยายเอกสารไฮเปอร์มีเดีย (Hypermedia Document Description Language) เพื่อนำเสนอเอกสารนั้น เผยแพร่ในระบบเครือข่ายใยแมงมุม WWW (World Wide Web) มีโครงสร้างการเขียน ที่อาศัยตัวกำกับ เรียกว่า แท็ก (Tag) ควบคุมการแสดงผลของข้อความ, รูปภาพ หรือวัตถุอื่นๆ ผ่านโปรแกรมบราวเซอร์ (Browser)

แท็ก

     แท็ก เป็นลักษณะเฉพาะของภาษา HTML ใช้ในการระบุรูปแบบคำสั่ง หรือการลงรหัสคำสั่ง HTML ภายในเครื่องหมายน้อยกว่า (<) และมากกว่า(>) โดยที่แท็ก HTML แบ่งได้ 2 ลักษณะคือ
           - แท็กเดี่ยว เป็นแท็กที่ไม่ต้องมีการปิดรหัส เช่น <P>, <HR> เป็นต้น
           - แท็กเปิด/ปิด เป็นแท็กที่ประกอบด้วย แท็กเปิด และ แท็กปิด โดยแท็กปิด จะมีเครื่องหมายสแลช (/) นำหน้าคำสั่งในแท็กนั้นๆ เช่น <B>…</B>, <P>…</P> เป็นต้น

คุณลักษณะ

     เป็นส่วนขยายความสามารถของแท็กจะต้องใส่ภายในเครื่องหมาย < > ในส่วนแท็กเปิดเท่านั้นแท็กคำสั่ง HTML แต่ละคำสั่ง จะมีคุณลักษณะ แตกต่างกันไป และมีจำนวนไม่เท่ากัน การระบุคุณลักษณะ มากกว่า 1 คุณลักษณะ ให้ใช้ช่องว่างเป็นตัวคั่น เช่น แท็ก ควบคุมเกี่ยวกับรูปภาพ <IMG> มี Attributes ดังนี้
           <IMG SRC="ชื่อไฟล์.นามสกุล" WIDTH="n" HEIGHT="n" ALT="text" BORDER="n">
                - SRC เป็นคุณลักษณะควบคุมชื่อไฟล์ภาพที่จะนำมาแสดงผล
                - WIDTH เป็นคุณลักษณะควบคุมความกว้างของภาพ
                - HEIGHT เป็นคุณลักษณะควบคุมความสูงของภาพ
                - ALT เป็นคุณลักษณะควบคุมคำอธิบายภาพ
                - BORDER เป็นคุณลักษณะควบคุมขนาดของเส้นขอบของภาพ

ไฟล์เอกสาร HTML เป็นไฟล์ข้อความรูปแบบหนึ่ง (Text File) ที่เก็บชุดคำสั่ง HTML ดังนั้นการสร้างเอกสาร HTML จึงสามารถใช้ Text Editor ตัวใดก็ได้ เช่น NotePad Editplus หรือ Dreamweaver

แสดงผลลัพธ์
     แต่ละข้อความที่พิมพ์มานี้เราเรียกมันว่าแท็กซึ่งแท็กแรกเป็นแท็กที่ใช้บอกว่าเป็นการเริ่มต้นเอกสาร และต่อมาก็คือแท็กที่ใช้บอกจุดสิ้นสุด การกำหนดว่าแท็กใดเป็นแท็กสิ้นสุด สามารถทำได้โดยการใส่เครื่องหมาย "/" ไว้ที่หน้าแท็กเป็นส่วนใหญ่ , แต่ก็ไม่ได้หมายความว่าทุกแท็ก จำเป็นจะต้องมีแท็กปิดเสมอไป เมื่อเรานึกถึง "แท็ก" ก็เหมือนกับว่า เราได้พูดคุยกับ โปรแกรมเบราส์เซอร์ หรือไม่ก็เป็นการบอกรายละเอียดให้กับมัน แล้วอะไรละที่จะบอกสิ่งแรกก็คือ "นี่คือการเริ่มต้นเอกสาร <HTML> และสิ่งต่อมาก็คือ "นี่คือการสิ้นสุดเอกสาร" </HTML> เมื่อมาถึงตรง นี้แล้ว สิ่งที่จะต้องทำต่อไปก็คือเริ่มใส่แท็กอื่น ๆ เข้าไปอีก ในเอกสาร HTML โดยทั่วไปมักจะต้องมีแท็ก HEAD เพื่อกำหนดรายละเอียดในส่วนหัวของเอกสาร

<html>
     <head>
     </head>
</html>

แสดงผลลัพธ์
     และรายละเอียดแรกที่เราจะใส่ไว้ในส่วนหัวของเอกสาร (อยู่ระหว่างแท็ก HEAD) ก็คือ แท็กไตเติ้ล TITLE ซึ่งก็คือชื่อเรื่องนั่นเอง

<html>
     <head>
          <title><title>
     </head>
</html>

แสดงผลลัพธ์
     และต่อมาก็คือแท็ก BODY ซึ่งเป็นแท็กที่สำคัญอย่างยิ่ง เพราะต่อไปรายละเอียดต่างๆ ใน เอกสารของเราจะถูกใส่ไว้ภายในแท็กนี้

<html>
     <head>
          <title><title>
     </head>
     <body>
     </body>
</html>

แสดงผลลัพธ์
     อีกสิ่งหนึ่งที่เราจะทำกันก็คือ, ใส่ชื่อหรือไตเติ้ลให้กับเอกสาร

<html>
     <head>
          <title>ยินดีต้อนรับสู่การเขียนเว็บเพจด้วยภาษา HTML<title>
     </head>
     <body>
     </body>
</html>
